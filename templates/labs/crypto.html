Objetivo

Inicia sesión y revisa la cookie auth_token: es un MD5 sin sal de la contraseña. Crackea el hash, recupera la contraseña y vuelve a entrar para obtener la flag.

Pasos de explotación

  1. Inicia sesión con cualquier usuario (ej. alice / password).
  2. Abre DevTools → Application → Cookies y copia el valor de auth_token.
  3. El token es el hash MD5 sin salt de la contraseña.
  4. Crackea el hash con hashcat o john:
echo "2ac9cb7dc02b3c0083eb70898e549b63" > hash.txt
hashcat -a 0 -m 0 hash.txt /usr/share/wordlists/fasttrack.txt
john --format=raw-md5 --wordlist=/usr/share/wordlists/fasttrack.txt hash.txt
  1. También puedes capturar la respuesta HTTP con Burp Suite para ver el hash en el cuerpo de la respuesta.
  2. Hashes de muestra: alice / Password1 → 2ac9cb7dc02b3c0083eb70898e549b63
